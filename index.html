phiori
우카가카를 위한 python3 기반의 SHIORI


첫번째 배포 버전을 다운로드 개발 단계의 최신 소스 코드를 다운로드 릴리즈 목록

시작하기


phiori 배포 버전은 페이지 상단에서 다운로드할 수 있습니다.

phiori 배포 버전을 제작할 고스트 폴더에 아래와 같이 넣어주세요.

home/
  +-ghost/
    +-myghost/
      +-ghost/
        +-master/
          +-phiori/
          +-lib.zip
          +-phiori.dll
          +-python33.dll

이후 일반적인 고스트 작성 방법에 따라 descript.txt 파일을 작성합니다. 이 때, shiori는 phiori.dll으로 지정합니다.

shiori,phiori.dll

틀


descript.txt가 있는 곳과 같은 곳에 python3 스크립트 파일을 작성하여 곧바로 고스트를 만들 수 있습니다.

확장자가 .py인 python3 스크립트 파일을 만들어 직접 코드를 입력하거나, 아래 틀을 수정하여 사용하세요.

예시


phiori를 이용해 고스트를 만드는 기본적인 예시를 몇 개 준비했습니다.

기동 시 인사

@handle("OnBoot")
def boot(**args):
	print(r"\1\s0\0\s0여어. \w4잘 지냈어?\e")

더블클릭 시 메뉴 보이기

@handle("OnMouseDoubleClick")
def mousedoubleclick(**args):
	print(r"\0\s0")
	print(random.choice(["뭔데.", "응?", "왜."]))
	print(r"\w4\n\n")
	print(r"\![*]\q[뭔가 말해봐,OnTalk]\n")
	print(r"\![*]\q[아무것도 아니야,cancel]\n")
	print(r"\e")

랜덤 토크

@handle("OnTalk")
def talk(**args):
	food = random.choice(["붕어빵", "고기만두", "타코야키"])
	print(random.choice([
		r"\1\s0\0\s4후우…\n\n\w9\1무슨 일이야?\n\n\w9\0\s0갑자기 {} 먹고 싶어.\n\n\w9\1%username에게 사달라고 해.\n\n\w9\0\s8이녀석은 분명 사들고 와서 내 눈 앞에서 전부 먹어치울 거야. \w6틀림 없어.\n\n\w9\1…….".format(food),
		r"\1\s0\0\s7…\w8…\w8.\n\n\w4\w8\w9\1뭐…뭐야, \w4무섭잖아.\n\n\w9\0내가 사 놓은 {}, 네가 먹었냐.\n\n\w9\1…\w4…\w4.\n\n\w9\0…\w4…\w4.\n\n\w9\1…\w4…\w4.\n\n\w9\0…\w4…\w8뭔데, 없애버린다.\n\n\w2\1제, \w4제가 잘못했습니다…\n\n\w9\0…\w4…\w8\s4후우…".format(food),
	])
	print(r"\e")

쓰다듬기 판정

@handle("OnStroke")
def stroke(**args):
	if args["Reference1"] == "Head":
		print(r"\0\s1…\w4…\w8기분은 \w4좋네…\e")


독자 이벤트


작업의 편의를 위해 phiori에서는 몇 개의 독자 이벤트를 제공합니다.

OnTalk

랜덤 토크 이벤트입니다. 사용자가 임의로 부르지 않을 경우엔 시간이 흐름에 따라 스스로 발동합니다.

인수 없음.

OnStroke

쓰다듬기 이벤트입니다. 판정 부위를 어느 정도 쓰다듬으면 발동합니다.

Reference0
쓰다듬은 대상
Reference1
쓰다듬은 판정 부위
Reference2
쓰다듬기 포인트
