시작하기


phiori 배포 버전은 페이지 상단에서 다운로드할 수 있습니다.

phiori 배포 버전을 제작할 고스트 폴더에 아래와 같이 넣어주세요.

home/
  +-ghost/
    +-myghost/
      +-ghost/
        +-master/
          +-dlls/
          +-libs/
          +-phiori/
          +-msvcr100.dll
          +-msvcr110.dll
          +-phiori.dll
          +-python33.dll

이후 일반적인 고스트 작성 방법에 따라 descript.txt 파일을 작성합니다. 이 때, shiori는 phiori.dll으로 지정합니다.

shiori,phiori.dll

예시


phiori를 이용해 고스트를 만드는 기본적인 예시를 몇 개 준비했습니다.

기동 시 인사

@handle("OnBoot")
def boot(**args):
	write(r"\1\s0\0\s0여어. \w4잘 지냈어?\e")

더블클릭 시 메뉴 보이기

@handle("OnMouseDoubleClick")
def mousedoubleclick(**args):
	write(r"\0\s0")
	write(["뭔데.", "응?", "왜."])
	write(r"\w4\n\n")
	write(makemenu(OnTalk="뭔가 말해봐", cancel=random.choice(["아무것도 아니야.", "심심해서 불러봤어."])))
	write(r"\e")

랜덤 토크

@handle("OnTalk")
def talk(**args):

		r"\1\s0\0\s7…\w8…\w8.\n\n\w4\w8\w9\1뭐…뭐야, \w4무섭잖아.\n\n\w9\0내가 사 놓은 {}, 네가 먹었냐.\n\n\w9\1…\w4…\w4.\n\n\w9\0…\w4…\w4.\n\n\w9\1…\w4…\w4.\n\n\w9\0…\w4…\w8뭔데, 없애버린다.\n\n\w2\1제, \w4제가 잘못했습니다…\n\n\w9\0…\w4…\w8\s4후우…",

@handle("OnHourChange")
def hourchange(**args):
	hour = datetime.datetime.today().hour
	if hour == 0:
		return r"\1\s0\0\s0자정이네.\n\n\w8\1슬슬 졸리지 않아?\n\n\w8\0않아.\n\n\1…\w4…\w4.\e"
	if hour == 12:
		return r"\1\s0\0\s0정오네.\n\n\w8\1점심밥 먹자 점심밥.\n\n\w8\0잘래.\n\n\1…\w4…\w4.\e"


제공되는 기본적인 요소


작업의 편의와 확장성을 위해 phiori에서는 전역 사전에 몇 가지 요소를 기본적으로 제공합니다.

기본적으로 포함되는 모듈

  • datetime
  • json
  • os
  • random
  • re
  • sys
  • time
  • urllib

기본적으로 제공되는 메소드

void print(*objects, sep=" ", end=r"\n")

sep로 구분되며 끝에 end가 붙도록 문자열화된 objects를 베이스웨어에 출력값으로서 전송합니다. 필요한 인수는 키워드 인수로서 취급해야 합니다.

void write(text, *args, **kwargs)

print(text.format(*args, **kwargs), end="")의 간단한 표현입니다.

void writeline(text, *args, **kwargs)

print(text.format(*args, **kwargs))의 간단한 표현입니다.

기본적으로 제공되는 변수


phiori에서 제공하는 모든 변수는 phiori 개체를 통해 사용할 수 있습니다.

dict phiori.config

config.txt로부터 읽어들인 설정을 제공합니다. 형식은 문자열로 고정되며, 변경 사항은 phiori가 정상적으로 언로드될 때 저장됩니다.

readonly str phiori.encoding

python3가 사용하는 기본 인코딩을 제공합니다. sys.getdefaultencoding()와 같습니다.

readonly str phiori.locale

현재 시스템의 기본 로케일을 제공합니다. locale.getdefaultlocale()와 같습니다.

readonly dict phiori.info

베이스웨어가 NOTIFY해 보낸 모든 값을 제공합니다.

readonly str phiori.path

phiori.dll이 있는 디렉토리 위치를 제공합니다.

readonly dict phiori.res

베이스웨어로부터 리소스를 요청받았을 때 보내는 resource.txt로부터 읽어들인 리소스를 제공합니다.

dict phiori.var

variable.dat로부터 읽어들인 고스트 변수 사전을 제공합니다. 형식을 보존하며, 변경 사항은 실시간으로 저장됩니다.

dict phiori.words

words.dic로부터 읽어들인 어휘 사전을 제공합니다. 형식을 보존하며, 변경 사항은 phiori가 정상적으로 언로드될 때 저장됩니다.

미리 구성


기본적으로 제공되는 몇 가지의 변수를 사용하기 위해 미리 만들어 사용할 수 있는 파일을 제공합니다.

config.txt

phiori에서 읽고 쓸 수 있는 설정 보존 파일입니다. 형식은 descript.txt 스타일을 따르며, 모든 값은 텍스트로서 취급됩니다.


config.txt

talk.interval,60
temae,1
debug,1

Python3 스크립트

print(phiori.config["talk.interval"])
print(phiori.config["temae"])
print(phiori.config["debug"])
phiori.config["talk.interval"] = 60
phiori.config["temae"] = 1
phiori.config["debug"] = 1

resource.txt

phiori에서 NOTIFY 요청을 받았을 때 반환하는 값이 들어있는 읽기 전용 리소스 파일입니다. 형식은 descript.txt 스타일을 따르며, 모든 값은 텍스트로서 취급됩니다.


resource.txt

homeurl,http://ukagaka.example.com/ghost/
vanishbuttonvisible,0

Python3 스크립트

print(phiori.res["homeurl"])
print(phiori.res["vanishbuttonvisible"])

words.dic

phiori에서 읽고 쓸 수 있는 어휘 사전 파일입니다. 형식은 JSON이며, 모든 값은 형식이 유지됩니다.


words.dic

{
    "name": [
        "사쿠라",
        "케로"
    ],
    "food": [
        "붕어빵",
        "사과",
        "포키"
    ]
}

Python3 스크립트

print(phiori.words["name"][0])
print(random.choice(phiori.words["food"]))
phiori.words["name"] = [
	"사쿠라",
	"케로"
]
phiori.words["food"] = [
	"붕어빵",
	"사과",
	"포키"
]

편리한 기능


작업의 편의를 위해 사쿠라 스크립트를 대체하는 몇 가지의 메소드를 제공합니다.

str makemenu(*args)
str makemenu(**kwargs)

메뉴를 나타내는 사쿠라 스크립트를 만듭니다. 위치 의존 인수일 경우 선택지 ID와 표시 이름이 같은 메뉴가 생성되고, 키워드 인수일 경우 파라메터 이름이 키의 문자열 표현이고, 표시 이름이 값의 문자열 표현인 메뉴가 생셩됩니다.

makemenu("메뉴1", "메뉴2", "메뉴3")
makemenu(menu1="메뉴1", menu2="메뉴2", menu3="메뉴3")

str makemenuitem(title)
str makemenuitem(title, id, *args)

title로 표시되는 메뉴 항목을 나타내는 사쿠라 스크립트를 만듭니다. id를 넘겨받았다면 선택지 ID가 id로 지정됩니다. id가 On 이벤트라면 args가 이벤트의 변수로서 넘어갑니다.

makemenuitem("취소")
makemenuitem("랜덤 토크", "OnTalk")
makemenuitem("상세 메뉴", "OnMenuSelect", "alternative")

str say(narrator, text=None)
str say(narrator, surface, text)

narrator가 text라고 말하는 사쿠라 스크립트를 만듭니다. surface가 주어진 경우 지정된 서페이스를 표시하게 합니다.

say(1, "나는 케로야.")
say(0, 5, "나는 사쿠라야!")
